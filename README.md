使用bootstrap，直接进行简单的登录和注册布局。再通过按钮来进行页面跳转和登录请求。

![p1](p1.png)

如果请求成功，先跳转成功页面，

![p2](p2.png)

反之跳转失败页面，并且返回失败原因

![p3](p3.png)

跳转首页

![p4](p4.png)

右上角是添加按钮，图表记录按钮，退出按钮，中间是我们设置账单记录，下面是通过js写的分页，每四个账单一页

账单就显示时间，标题，留言，花费类型，金额，后面还有删除和修改按钮，都是直接使用的bootstrap样式和图标

 

这是修改页面，简单的bootstrap样式，可以直接修改数据

 ![p5](p5.png)
